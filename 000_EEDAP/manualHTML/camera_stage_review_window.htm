eeDAP: Evaluation Environment for Digital and Analog Pathology,     Camera Stage Review window

Camera Stage Review window

Take image button

User could use this button to take image of the current camera review. If user chosen stage control system, the default image name has stage position information. Otherwise, the default name is in format "cam_+number". User could also edit the image name and saving directory.

Choose system

User could choose his working system at here. Without choose a system "Stage current position" and "Stage destination position" would be disabled. But user could still use Take image button to get camera image
